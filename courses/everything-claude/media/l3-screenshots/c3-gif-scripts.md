## GIF Scripts

GIF SCRIPT 1: Model Selector Tour
- Duration: ~8 seconds
- Setup: claude.ai open, new conversation, model selector closed
- Actions:
  1. Click model selector to open dropdown
  2. Hover over Haiku 4.5 briefly (2 seconds)
  3. Hover over Sonnet 4.5 briefly (2 seconds)
  4. Hover over Opus 4.6 briefly (2 seconds)
  5. Select Opus 4.6
  6. Pause to show Opus selected (2 seconds)
- What the viewer sees: All three models in the dropdown and how to switch between them
- Suggested filename: l3-model-selector-tour.gif

GIF SCRIPT 2: Extended Thinking in Action
- Duration: ~15 seconds
- Setup: New conversation with Sonnet 4.5 or Opus 4.6 selected
- Actions:
  1. Type prompt: "Explain why the sky is blue using physics"
  2. Pause to show prompt typed (2 seconds)
  3. Press Enter/Send
  4. Watch for thinking indicator to appear
  5. Wait for thinking summary to become visible
  6. Pause on the visible thinking summary (3 seconds)
- What the viewer sees: The full thinking process from prompt to visible reasoning
- Suggested filename: l3-extended-thinking-action.gif

GIF SCRIPT 3: Haiku Speed Demo
- Duration: ~8 seconds
- Setup: New conversation with Haiku 4.5 selected
- Actions:
  1. Show Haiku selected in model selector (1 second)
  2. Type a simple prompt: "What's the capital of New Zealand?"
  3. Send the message
  4. Watch near-instant response stream in
  5. Pause on completed response (2 seconds)
- What the viewer sees: How fast Haiku responds to a simple question
- Suggested filename: l3-haiku-speed-demo.gif
